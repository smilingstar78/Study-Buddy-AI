## 🌟 <span style="color:#4CAF50;">Overview</span>

<p>
<b>Study Buddy AI</b> is your personal AI tutor designed to simplify learning and make education feel effortless.  
It understands your questions, breaks down complex topics, and gives clear explanations like a friendly mentor 💡
</p>

## 🛠️ Installation

````
# Clone repo
git clone https://github.com/your-username/study-buddy-ai.git

# Go inside project
cd study-buddy-ai

# Install dependencies
pip install -r requirements.txt

# Run project
python app.py
````
